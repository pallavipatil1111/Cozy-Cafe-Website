# Cozy-Cafe-Website
Description:
A responsive and visually appealing multi-page café website designed using HTML, CSS, and JavaScript to showcase a modern café menu and information.

Features:
Home page with welcome section and café overview

Menu page displaying available food and beverages

Contact form (front-end only)

Animated effects and hover transitions

Fully responsive layout for desktop and mobile

Files:
index.html – Home page

menu.html – Café menu

details.html – Café details/about section

style.css – Styling and responsiveness

script.js – Handles animations and interactions

How to View:
Open index.html in any browser to explore the full website.

(Optional) Host on GitHub Pages, Netlify, or Vercel for live access.

Notes:
This project demonstrates front-end development skills, including responsive web design, layout structuring, and interactive UI creation.
